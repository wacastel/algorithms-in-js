# algorithms-in-js

Algorithms in JavaScript

Starting the server:

npx http-server -c-1 -p 8080

Notes:
-use the -c-1 parameter to disable the browser cache (using a value of -1)
-use p 8080 to specify the TCP port 8080

Go to chrome and load the page:
http://localhost:8080/
